# Documentation Templates

Use these reusable templates to keep new content consistent across the knowledge base.

## Included Templates

- Knowledge article
- Interview question
- Bug case study
- Resume note
- Company research
- Learning note

## Shared Metadata Pattern

All templates use the same front matter structure so pages are easy to filter, review, and maintain.

```yaml
---
title: ""
summary: ""
status: draft
tags: []
owner: ""
created: YYYY-MM-DD
updated: YYYY-MM-DD
---
```

## Usage

1. Copy the template that best matches the content type.
2. Fill in the front matter before writing the page body.
3. Keep the section headings whenever possible so the documentation stays predictable.
