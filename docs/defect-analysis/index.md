# Defect Analysis

Understanding failures and preventing recurrence.

A bug found in production is information. The value is not just fixing the symptom — it is understanding why the defect existed, why it was not caught earlier, and what changes prevent the next one. These articles document the analytical process that turns individual bugs into systemic improvements.

## Articles

- **[Root Cause Analysis](root-cause-analysis.md)** — Going beyond the surface failure to the underlying engineering decision that allowed it
- **[Incident Retrospectives](incident-retrospective.md)** — Running blameless post-mortems that produce durable process improvements
- **[Regression Prevention](regression-prevention.md)** — Closing the feedback loop so fixed bugs stay fixed
