# Python & Pytest

Test engineering fundamentals and advanced patterns.

pytest is the backbone of most Python-based test suites, but its full power is underused. These articles cover practical patterns: fixtures that reduce duplication, parametrize that scales to hundreds of cases, and plugins that make test output actionable.

## Articles

- **[Fixture Design](fixture-design.md)** — Building reusable, composable test setup without hidden dependencies
- **[Data-Driven Testing](data-driven-testing.md)** — Scaling test coverage with parametrize and external data sources
- **[Custom Reporters](custom-reporters.md)** — Producing test output that stakeholders can read and act on
- **[Testing Patterns](testing-patterns.md)** — AAA structure, test isolation, and patterns that prevent false positives
