# Appium & Mobile

iOS and Android test architecture at scale.

Mobile testing introduces challenges that web testing does not: platform divergence, device variability, OS version fragmentation, and provisioning complexity. These articles document approaches that work in production — not just on a developer's local simulator.

- **[Shared iOS/Android Architecture](shared-architecture.md)** — Designing one test framework that runs across both platforms
- **[Device Farm Integration](device-farm.md)** — Running tests at scale on real devices without managing physical hardware
- **[Mobile Test Challenges](mobile-challenges.md)** — Handling flakiness, OS version gaps, and deep link testing
